## Reflection Struggling/Solved

- **Borrow checker with PagedKVCache**: `write_kv`/`read_kv` needed careful separation of page table index lookup from page data mutation. Solved by extracting page index first, then indexing into `self.pages`.
- **GQA backward compat**: When `n_kv_head == n_head`, `kv_dim == n_embd` and `kv_group_offset == q_head_offset` — mathematically identical to old MHA. Verified with deterministic seed tests.
- **REST feature isolation**: All async code behind `#[cfg(feature = "rest")]`. `merge_retrieved_branches` is pure computation (no feature gate) so it can be tested without REST dependencies.
- **anyrag API mismatch**: Plan assumed raw vector POST, but actual anyrag `/search/vector` takes text queries. Added `embedding_to_query()` to quantize hidden state into searchable text tokens.
