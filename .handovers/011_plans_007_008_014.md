## Reflection: Struggling / Solved

### Struggles
1. **wgpu API version**: `wgpu = "23"` has breaking changes from tutorials — `Instance::new()` takes `InstanceDescriptor` by value (not `&`), and `request_device()` requires a `trace_path: Option<&Path>` second argument. Had to fix these.
2. **Serde with HashMap**: `HashMap<String, usize>` needs custom serde for portable serialization. Used `map_serde` module to serialize as `Vec<(String, usize)>`.
3. **`ConstraintPruner::is_valid` takes `&self`**: Can't mutate `PartialParser` in the trait impl. Created a fresh `PartialParser` per call — acceptable since Tier 0 is O(n) and cheap.

### Solved
1. **Path encoding 5→16 bit**: Straightforward search-and-replace, but had to be thorough — `<< 5` appears in multiple contexts (path construction, parent extraction, sibling seeding, merge branches).
2. **Paged attention**: Used temporary flat KV buffers per layer to reuse existing `attention_head` kernel, avoiding the need to rewrite attention for page-by-page reads.
3. **Feature flag coexistence**: `validator` and `gpu` features are independent and compile cleanly together. `serde`/`serde_json` moved from optional (rest-only) to non-optional since tokenizer needs them.
