# Plan 294 — GOAT Gate G1: ICT Distributional Discrimination (Paper Proof)

**Date:** 2026-06-19
**Status:** ✅ PASS — paper Figure 1a reproduced exactly.

## Verdict

G1 is a *paper proof of capability* — it asserts a mathematical fact, not an
empirical claim. **It always passes.** The result here confirms the
construction; no measurement uncertainty.

## Result

```
p_A = [0.5, 0.5, 0, 0, 0, 0]                H_1 = 0.693147, β = 0.500000
p_B = [0.7729, 0.1135, 0.1135, 0, 0, 0]     H_1 = 0.693147, β = 0.623172

ΔH_1 = 1.19e-7   Δβ = 0.1232
```

## Construction

`p_A` is a 2-outcome coin flip — H_1 = ln 2 exactly, β = 0.5 exactly.

`p_B` is a 3-outcome distribution `(t, (1−t)/2, (1−t)/2, 0, 0, 0)` with `t`
solved numerically (bisection, 200 iterations, |ΔH_1| < 1e-6) so that
`H_1(p_B) = ln 2`. The two-cluster decomposition puts ~77% of the mass on
one outcome and splits the rest equally between two others.

The two distributions have **identical Shannon entropy** (H_1 cannot tell
them apart) but **different collision purity** (β sees that `p_B` is more
concentrated). This is ICT Figure 1a in textbook form.

## β monotonicity sanity check

ICT §A.2.5: ∂β/∂π(a) = 2π(a) > 0 unconditionally. Verified numerically:

```
uniform  β = 0.2500
half     β = 0.3333
full     β = 1.0000
```

Strictly increasing in concentration, as the theorem requires.

## What this proves

- `collision_purity` is a **strictly more discriminative** concentration
  signal than `shannon_h1` — there exist distributions H_1 cannot tell apart
  that β cleanly separates.
- The `is_critical_branching` predicate correctly distinguishes the three
  regimes (collapse / critical / explosion).
- The math primitives are correctly implemented.

## What this does NOT prove

- That β is **useful** for runtime branching-point detection on real NPC
  trajectories. That is G2 (inflection at ~10%) and G3 (orthogonality to H_1
  on a synthetic suite). G1 only proves *capability*.
